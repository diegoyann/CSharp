# Sales System

### This is a Sales System developed by me while I was coursing the C# Developer course at Udemy.

## Used concepts
- Object Oriented programming 
- .Net C#
- Classes, attributes, methods and static members
- Constructors
- Enumerations and compositions
- Inheritance and polymorphism
- Interfaces
- Generics, set and dictionarys
- Lambda expresions, delegates and LINQ
